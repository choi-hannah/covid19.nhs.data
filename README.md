# Trust-level COVID-19 hospitalisations in England


## Summary

This repo contains:

1. Trust-level hospital admissions data, published weekly on the NHS [COVID-19 Hospital Activity](https://www.england.nhs.uk/statistics/statistical-work-areas/covid-19-hospital-activity/) page.
2. A many-to-many mapping between lower-tier local authority districts and NHS Acute Trusts in England; details can be found in [reports/mapping_summary](https://github.com/sophiemeakin/covid19-uk-hospitalisation-data/blob/main/reports/mapping_summary.md).
